# 3/22/2026 3 PM - Implementing the Design to Connect the Wearable Tech

_Time spent: 5.5h_

I debated scrapping the USB webcam for a Pi Camera, but realized the ribbon cables are too fragile for a wearable. Decided to stick with USB but move to a mini-module for better weight distribution on the glasses.
Made the final design for the complete processor that can be directly strapped onto the person through any strap or a harness.
